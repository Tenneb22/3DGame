## 3D Game
This 3D game is written in Java with the LWJGL 3.2.3 library.

The game is based of the [tutorial series by DevGenie](https://youtube.com/playlist?list=PL4QbKfRYvHvSJSVgHawYIvcIA0CcgnMzw&si=gkaxEFmmGpvmtE1J)
